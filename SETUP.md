# Siamese Cat Adventure - Setup Instructions

## Game Files Created

The following files have been created for your game:
- `index.html` - Main game page
- `style.css` - Game styling
- `game.js` - Game logic and mechanics

## Required Images

To complete the game setup, you need to add three PNG image files to this directory:

1. **image1.png** - The Siamese cat (player character)
2. **image2.png** - The blue monster (enemy)
3. **image3.png** - The gold coin (collectible)

## How to Add the Images

1. Save your three PNG images with the exact names listed above
2. Place them in the same directory as index.html
3. Open index.html in a web browser to play the game!

## Game Controls

- **W** - Jump
- **A** - Move Left
- **D** - Move Right

## Game Objective

- Collect all 10 coins before time runs out (90 seconds)
- Avoid the blue monsters
- Special items will appear after collecting certain coins

## Features

- Platformer physics with jumping and gravity
- Moving enemies that patrol platforms
- Collectible coins
- Timer countdown
- Item system with power-ups:
  - Speed Boost
  - Double Jump
  - Shield
  - Time Freeze
- Dialog system for item selection
- Win/lose conditions

Enjoy your Siamese Cat Adventure!
